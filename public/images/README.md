# Website Images

This directory contains all the images used on the website, organized into subfolders based on the page or section where they appear.

To update an image on the website, simply replace the corresponding file in one of the subfolders with your new image. **Make sure the new image has the exact same file name.**

## Folder Structure

- **/public/images/about/**
  - `story.png`: Image for the "Our Story" section on the About page.

- **/public/images/cloud-x/**
  - `workflow.png`: Diagram for the "How It Works" section on the Cloud-x page.

- **/public/images/home/**
  - `hero-image.png`: The main hero image on the homepage.

- **/public/images/products/**
  - `vps.png`: Image for the Virtual Private Servers section.
  - `dedicated.png`: Image for the Dedicated Servers section.
  - `shared-service.png`: Image for the Shared Service (Cloud-x.in) section.
  - `web-hosting.png`: Image for the Web Hosting section.
  - `colocation.png`: Image for the Server Colocation section.
  - `on-premise.png`: Image for the On-premise Cloud section.

You do not need to make any code changes. Just upload your new images to the correct folder with the correct name.
